# cc-deck

A collection of Claude Code plugins by darknight.

## Plugins

| Plugin | Description |
|--------|-------------|
| [deck](plugins/deck/) | A developer's skill deck for Claude Code - streamlined workflows for git commits, PRs, and task management |

## Installation

### Add Marketplace

```bash
# In Claude Code
/plugin marketplace add darknight/cc-deck

# Or in Terminal
claude plugin marketplace add darknight/cc-deck
```

### Install Plugins

```bash
# In Claude Code
/plugin install deck@cc-deck

# Or in Terminal
claude plugin install deck@cc-deck
```
